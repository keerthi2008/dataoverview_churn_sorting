# 📊 Customer Churn Analysis (Excel Project)

## 📌 Project Overview

This project focuses on analyzing customer churn data using Microsoft Excel. The goal is to identify patterns and factors that contribute to customer churn and provide actionable insights.

---

## 🎯 Objectives

* Understand customer behavior
  
* Identify churn patterns
  
* Analyze key factors affecting churn
  
* Build an interactive Excel dashboard

---

## 📂 Dataset

* Telco Customer Churn Dataset
  
* Contains customer details like:

  * Gender
    
  * Tenure
    
  * Monthly Charges
    
  * Services
    
  * Churn Status

---

## ⚙️ Tools Used

* Microsoft Excel
  
* Pivot Tables
  
* Charts & Graphs
  
* Data Cleaning Techniques

---

## 🔍 Process


### 1. Data Cleaning

* Removed duplicates
  
* Handled missing values
  
* Converted data types
  
* Standardized text values


### 2. Data Analysis


* Sorted data by tenure
  
* Filtered churn customers
  
* Created pivot tables

### 3. Dashboard Creation

* KPI Cards:

  * Total Customers
    
  * Churn Rate
    
  * Average Monthly Charges
    
* Charts:

  * Churn Distribution
    
  * Services vs Churn
    
  * Tenure Analysis

---

## 📊 Key Insights

* Customers with low tenure are more likely to churn
  
* High monthly charges increase churn probability
  
* Lack of services like Tech Support leads to higher churn

---

## 💡 Recommendations

* Improve onboarding for new customers
  
* Offer discounts for high monthly charges
  
* Promote value-added services

---

## 🚀 Conclusion

This project demonstrates how Excel can be used for data analysis and business insights generation.
